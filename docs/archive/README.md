# Archive Documents

이 디렉토리는 과거 조사 자료와 초안 문서를 보존하는 용도입니다. 아래 문서는 삭제하지 않고 남겨두지만, 현재 구현 기준 문서로 직접 사용하지 않습니다.

archive 문서에는 예전 브랜치 전략, 대안 기술 스택, 초안성 API 표현이 남아 있을 수 있습니다.

## 보존 문서

- `구미_AR_부동산_프로젝트_통합문서.md`
  - 장문의 시장 조사, 기술 조사, 발표용 서술, 대안 비교가 포함된 원천 문서
- `프로젝트_초기_세팅_가이드.md`
  - 초기 세팅 가이드와 장기 일정 초안
